# Conversia AI Website

AI-powered coaching platform for healthcare technology teams.

## Quick Start

1. **Deploy to AWS S3**: Follow `DEPLOY_GUIDE.md` for step-by-step instructions
2. **Full AWS Setup**: See `AWS_GITHUB_DEPLOYMENT.md` for complete CloudFront + Route 53 setup
3. **Website**: Open `index.html` to preview locally

## Files Included

- `index.html` - Complete website (9,426 bytes)
- `DEPLOY_GUIDE.md` - Quick AWS S3 deployment steps
- `AWS_GITHUB_DEPLOYMENT.md` - Full AWS architecture guide
- `.github/workflows/deploy.yml` - Automated deployment pipeline

## Live Website Features

- Responsive design optimized for all devices
- Purple (#4F2582) and brown (#B64621) branding
- Contact form with HubSpot integration
- Login link to staging environment
- Professional healthcare technology focus

## Deployment Architecture

- **S3**: Static website hosting
- **CloudFront**: Global CDN
- **Route 53**: DNS management
- **GitHub Actions**: Automated deployments
